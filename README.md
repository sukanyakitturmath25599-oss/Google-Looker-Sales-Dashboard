# Google-Looker-Sales-Dashboard
# 📊 Sales Dashboard – Google Looker Studio

## 📌 Project Overview
This project presents an interactive **Sales Analytics Dashboard** built using **Google Looker Studio**.  
The dashboard helps analyze business performance through key metrics such as sales, profit, customer behavior, and category performance.

It enables data-driven decision making by providing clear visual insights into trends, profitability, and operational performance.

---

## 🎯 Objectives
- Monitor overall sales performance
- Track net profit and revenue trends
- Analyze category-wise profitability
- Measure customer activity and order behavior
- Identify top performing products and segments
- Support business decision making through visual analytics

---

## 📈 Key Features
- KPI summary cards (Sales, Net Profit, Orders, Customers, AOV)
- Time-based trend analysis
- Category-wise profit analysis
- Customer-level performance insights
- Product performance tracking
- Payment method and transaction analysis
- Interactive filters (date, category, etc.)
- Dynamic visualizations and charts

---

## 🛠 Tools & Technologies
- **Google Looker Studio** – Data visualization
- **Data Source** – Sales dataset
- **Data Modeling** – Calculated metrics & aggregations
